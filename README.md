# Koach Web App

A secure Ionic + Angular coaching web app with login, pre-assessment, strength & aspiration conversations, report and insights pages, and feedback workflows.

## Features

- Login and authenticated user access
- Pre-assessment workflow
- Strength & aspiration conversation module
- Report and insights dashboard
- Feedback request flows
- Capacitor-ready Ionic Angular mobile/web architecture

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app locally:
   ```bash
   npm start
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/` — application source code
- `src/app/` — Angular app modules, pages, and routing
- `src/assets/` — static assets
- `src/environments/` — environment configuration

## Notes

- This project uses Ionic 8, Angular 20, and Capacitor 8.
- `date-fns` is included for date handling.
